# Finland 1999 - 2007

## Metadata

**Startdate:** 1999-01-01\
**Enddate:**

## Description

The Finnish euro coins have three different designs. The coins from 1 euro cent to 50 euro cent have the same design. The 1 and 2 euro coin both have an unique design.
